VISA BOKNINGEN
Möt upp din städare

Vi rekommenderar dig att alltid möta upp din städare, i alla fall första gången så att ni träffas personligen. Bekräfta alltid att den person som kommer till dig är den som du bokat, annars gäller inte försäkringar etc.

Berätta vad du vill ha gjort

Du kan alltid chatta med din städare via BringClean vilket rekommenderas om det mot förmodan skulle uppstå meningsskiljaktigheter. Ha förståelse för vad en städare kan hinna med under den tid som du valt att betala för och ha en öppen dialog med varandra för bästa resultat.

Ha städmaterial på plats

Städarna har inte med sig eget material utan det är alltid upp till dig som kund att se till att du har allt material på plats för att städaren ska kunna göra ett bra jobb. Om du saknar material så erbjuder BringClean olika miljövänliga alternativ som du kan beställa genom att kontakta oss.
